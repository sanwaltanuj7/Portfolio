## 🚀 Local Setup

Clone the repository:

```bash
git clone https://github.com/sanwaltanuj7/Portfolio.git
```

Navigate to the project:

```bash
cd Portfolio
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```
